## [0.1.0] - 2026-01-28

### 🎉 Initial Release

- Initial release of etsa.tech website
- Next.js-based static site with blog functionality
- Integration with Google Sheets for event management
- Email notification system via Nodemailer
- Authentication with NextAuth
- Responsive design with Tailwind CSS
- Markdown support for blog posts with remark
- Code quality checks with ESLint
- Security scanning with CodeQL
- Performance monitoring with Lighthouse
- Accessibility testing with axe-core
